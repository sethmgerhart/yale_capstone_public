# Air Quality Monitor Representativeness in the Contiguous United States \-&gt; data \-&gt; **metadata**

## Description

**Project Objective:** Determine whether state and local governments site air monitors in areas that capture data that are representative of actual air quality. 

**Project Folder:** The metadata folder contains metadata for this project's final, published datasets. Information about intermediate datasets is contained in the notebooks in which the datasets were created. Raw dataset metadata is provided in the notebooks in which the datasets were first used.
