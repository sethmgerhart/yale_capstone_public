# Air Quality Monitor Representativeness in the Contiguous United States -> **Data**

## Description

**Project Objective:** Determine whether state and local governments site air monitors in areas that capture data that are representative of actual air quality. 

**Project Folder:** The Data folder contains the entire project, including the data, analyses, and results.

## Data Collection

- **Period**: January\-December 2019
- **Sites**: Contiguous United States
- **Frequency**: Daily
- **Instruments**: State and Local PM2.5 Air Monitoring Sites; Machine Learning Alternative PM2.5 estimates

## File Organization

  - `raw_data/` - Original files
    - Note: The contents of this folder are not uploaded to GitHub due to the size of the original files.
  - `processed_data/` - Cleaned intermediate files and final analysis results
  - `analysis/` - R scripts in a Jupyter Notebook
  - `metadata/` - Data documentation
  - `outputs/` - Figures and reports
  - `archive/` - Legacy Data

## Requirements

- R
- Packages: dplyr, tidyr, readr, vroom, ggplot2, stringr, terra
