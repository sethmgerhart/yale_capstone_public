# Air Quality Monitor Representativeness in the Contiguous United States \-&gt; C\_raw\_data \-&gt; **census**

## Description

**Project Objective:** Determine whether state and local governments site air monitors in areas that capture data that are representative of actual air quality. 

**Project Folder:** The census folder contains data derived from the U.S. Census Bureau's TIGER/Line® Shapefiles webpage. TIGER stands for the Topologically Integrated Geographic Encoding and Referencing system and represents the U.S. Census Bureau's geographic spatial data. TIGER data products are developed using the TIGER geospatial data as the primary source. 

**Source Link:** [https://www.census.gov/cgi\-bin/geo/shapefiles/index.php](https://www.census.gov/cgi-bin/geo/shapefiles/index.php) 

**Data Dictionary Link:** [https://www2.census.gov/geo/pdfs/maps\-data/data/tiger/tgrshp2022/TGRSHP2022\_TechDoc.pdf](https://www2.census.gov/geo/pdfs/maps-data/data/tiger/tgrshp2022/TGRSHP2022_TechDoc.pdf) 

## Data Collection

- **Period**: 2019
- **Sites**: United States; County and State Boundaries
- **Frequency**: Annual
- **Instruments**: Census Data

## File Organization

  - `county_boundaries/` - Folder containing the shapefile boundaries for every county in the United States 
  - `state_boundaries/` - Folder containing the shapefile boundaries for every state in the United States
