# Air Quality Monitor Representativeness in the Contiguous United States \-&gt; data \-&gt; **alternative pm2.5 data**

## Description

**Project Objective:** Determine whether state and local governments site air monitors in areas that capture data that are representative of actual air quality. 

**Project Folder:** The Air Monitors folder contains the scripts for one analysis. 

- The analysis \(D\) cleans and calculates the median alternative PM2.5 data in every contiguous U.S. county for every day in 2019. 

_Note: Five other intermediary analyses are in the air\_monitors folder because they pertains to cleaning the data associated with the air monitors._

## Data Collection

- **Period**: January\-December 2019
- **Sites**: Contiguous United States 
- **Frequency**: Daily
- **Instruments**: Machine Learning Alternative PM2.5 estimates

## File Organization

- `D_alt_data_county_typical_daily_PM25_analysis` \- R script in a Jupyter Notebook used to clean and calculate the median alternative PM2.5 data in every contiguous U.S. county for every day in 2019
